**The Scenario:** 
On a specific server, for extra security, the Boss wants `sudo` to ask for the **Root Password**, not the **User's Password**.

**The Question:**
What is the specific **Defaults** flag you add to the sudoers file to change the authentication from the "Invoker" (you) to the "Target" (root)?

**The Answer :** 
```bash
Defaults rootpw 
# or
Defaults targetpw.
```
